### **🚀 Development & Deployment Steps**

#### **1️⃣ Installation**
```sh
git clone https://github.com/your-username/AccelerateAI.git
cd AccelerateAI
npm install
```

#### **2️⃣ Run Development Server**
```sh
npm run dev
```
App available at `http://localhost:3000`

#### **3️⃣ Build for Production**
```sh
npm run build
```

#### **4️⃣ Deploy to Firebase**
```sh
firebase deploy
```
